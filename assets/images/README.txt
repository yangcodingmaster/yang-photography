照片存放规则
============

一、文件夹命名
  每个系列一个文件夹，文件夹名字必须和 data.js 里的 id 字段完全一致。
  例如：data.js 里写 id: 'tokyo-2024'，对应的文件夹就叫 tokyo-2024/

二、文件命名
  cover.jpg   — 该系列的封面图（gallery 页展示）
  01.jpg      — 系列内的第一张照片
  02.jpg      — 第二张，以此类推
  （统一用数字编号，方便排序）

三、关于页个人照片
  直接放在 assets/images/ 根目录，命名为 profile.jpg

四、图片规格建议
  封面图：1200 x 900 px（4:3 比例），JPEG，质量 85
  系列照片：长边不超过 2400 px，JPEG，质量 85
  个人照片：600 x 800 px（3:4 竖版），JPEG，质量 85

五、换掉占位图的方法
  1. 把自己的照片按命名规则放进对应文件夹
  2. 打开 data.js
  3. 把 src 字段从 Unsplash 网址改成本地路径
     例如：'assets/images/tokyo-2024/01.jpg'
